泗川传统记忆翻翻乐

与邻里一起游玩：翻牌、记忆、配对！

人数：2–6人 时长：10–20分钟 难度：简单–中等

1. 游戏简介

泗川传统记忆翻翻乐包含与泗川城市文化及名胜有关的24对图卡。玩家通过翻牌寻找相同图案。 这不仅是训练记忆力的桌游，也是一款让邻里围坐、一起聊聊泗川自然、饮食与节庆的社交内容。

2. 组件

  • 图卡48张（24对）—— 南一带海水浴场、泗川海上缆车、卧龙山等
  • 卡背——韩国传统丹青纹样
  • 游戏指南

3. 准备

  1. 将全部卡牌充分洗混。
  2. 将卡牌背面朝上，按矩形阵列铺在桌面。
  3. 最近去过泗川的人先开始。

4. 规则

  1. 翻开两张—— 轮到你时，翻两张卡查看图案。
  2. 若相同—— 取走该对卡牌，并可继续多进行一回合。
  3. 若不同—— 将卡牌扣回原位，轮到下一位玩家。
  4. 游戏结束—— 全部卡牌配对完成后，配对最多者获胜。

5. 计分（可选）

行为得分
配对成功+2分
连续两次成功额外+1分
配对失败0分

6. 难度调整

  • 简易：使用8对（16张）
  • 标准：使用16对（32张）
  • 挑战：使用全部24对（48张）
可通过减少卡牌数量或提供短暂“预览时间”来适配不同玩家。

7. 邻里扩展玩法

  • 聊一聊：配对后分享与该地点/美食相关的故事与记忆。
  • 速度赛：在1分钟内尽量完成更多配对。
  • 协作模式：全员协力完成全部配对并记录用时。

8. 收纳与保管

  1. 将卡牌按对整理收好。
  2. 避光防潮，存放于干燥处。
注意：拿取时动作放缓，避免刮伤指尖或折损卡角。
